Quest for Thiccness

This is a simple GYM web-app that I made for myself because I am too cheap to buy a subscription and i fucking hate Excel. I don't know how the fuck you stumbled here but welcome!.

NOTE: This web-app only stores information in the current device!! progression is not kept between devices I dont support polygamy.

How to use the web-app

Selected the exercises you want and then click on the action button to create a new program
Fill out the form, in the sec and click save
Then you can navigate to My Programs to see your programs
NOW WE GYM, WE GO GYM!!!!

About the web-app

This web-app is written in Angular v15 and uses the PrimeNG v15 styling framework.
